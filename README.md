#  📩 Sistema de plantilla HTML para Email Corporativo

## 📌 ¿Qué es este proyecto?
En este proyecto he creado una plantilla de email para onsistems, pensada para enviarla a clientes como si fuera un pequeño folleto digital.

La idea era que cuando la empresa quiera presentar sus servicios por correo, el mensaje no sea solo texto plano, algo más visual y profesional.


## 🎯 ¿Qué he hecho exactamente?
He diseñado un email en HTML donde:

* Aparece el logo de la empresa.
* Se explican los servicios principales.
* Hay un botón para que el cliente pueda perdir presupuesto.
* Todo tiene los colores corporativos de onsistems.

## 🛠 ¿Cómo lo he construido?
Lo he hecho únicamente con HTML:

En los correos electrónicos no se pueden usar las mismas técnicas que en una web normal, asi que he tenido que organizar todo con tablas para que se vea bien cuando se envía por Gmail o por correo profesional.

Tambien he puesto los estilos directamente dentro de cada elemento para asegurar que no se descuadre al enviarlo.

## 💬 Mi experiencia haciendo este proyecto
Al principio pensaba que hacer un email era como hacer una página web normal, pero me di cuenta que no. 

He aprendido que los correos tienen muchas limitaciones y que hay que adaptarse para que se vean bien en todos los dispositivos.

Aunque es un proyecto sencillo, me ha servido para entender cómo funcionan realmente los emails profesionales por dentro.
